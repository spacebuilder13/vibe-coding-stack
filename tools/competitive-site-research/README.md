# competitive-site-research

Sitemap-driven crawl of a public site → **Obsidian-style markdown vault** (`00-meta`, `01-hubs`, `02-pages`, `03-articles`, optional `assets/ux-audit`) plus scripts to generate **MOC / IA** notes.

- **Agent entry:** [AGENTS.md](./AGENTS.md)
- **Persona + deliverable contract:** [../../prompts/competitive-research-agent.md](../../prompts/competitive-research-agent.md)
- **Cursor skill:** [../../.cursor/skills/competitive-site-research/SKILL.md](../../.cursor/skills/competitive-site-research/SKILL.md)

## Why

Supports **competitive research**: primary-source capture (what they ship on the web) separate from your strategy doc, with a path to **design / IA audits** via screenshots.

## Requirements

Python 3.11+ recommended; see `requirements.txt`.
